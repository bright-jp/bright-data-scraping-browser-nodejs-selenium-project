# Bright Data Scraping Browser Nodejs Selenium Project

[![Bright Data Promo](https://github.com/luminati-io/LinkedIn-Scraper/raw/main/Proxies%20and%20scrapers%20GitHub%20bonus%20banner.png)](https://brightdata.jp/)

<a href="https://codesandbox.io/p/devbox/github/luminati-io/bright-data-scraping-browser-nodejs-selenium-project?file=%2Findex.js" target="_blank" rel="noopener">CodeSandboxで開く</a> をクリックし、GitHubアカウントでサインインしてから、リポジトリをフォークして変更を開始してください。

このプロジェクトは、Selenium WebDriverを使用してWebサイトからデータをスクレイピングするために <a href="https://brightdata.jp/products/scraping-browser" target="_blank" rel="noopener">Bright Data Scraping Browser</a> を使用する方法をデモします。

## Getting Started

1. `index.js` の `YOUR_BRIGHT_DATA_SCRAPING_BROWSER_ENDPOINT` の値を、実際のBright Data scraping browser endpointに置き換えてください
2. `"https://example.com` をターゲットURLに置き換えてください
3. 依存関係をインストールするために `npm install` を実行してください
4. スクレイピングを開始するために `node index.js` を実行してください
